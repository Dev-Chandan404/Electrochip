## ✨ About the Project

> A **modern, clean, and fully responsive single-page HTML/CSS template** built for an electrical services company based in Punjab, India. Electrochip demonstrates how a professional business website can be crafted entirely with pure HTML5 and CSS3 — no JavaScript, no frameworks, no build tools.

This project is a masterclass in **layout-first thinking** — proving that visual hierarchy, responsive grids, and polished UI can all be achieved with nothing but clean markup and expressive stylesheets.

| | Feature | Description |

| 🧹 | **No JavaScript** | Entirely layout-driven — pure HTML & CSS only |

## 🚀 Getting Started

No installation. No build step. No dependencies whatsoever.

```bash
# Clone the repo
git clone https://github.com/Dev-Chandan404/Electrochip.git
cd Electrochip

# Open directly in your browser
open index.html
```

Or simply double-click `index.html`. That's it.
