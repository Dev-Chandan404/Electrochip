# ────────────────────────────────────────────

# ELECTROCHIP

# Static Electrochip UI (HTML + CSS Only)

# ────────────────────────────────────────────

> A pure layout project built using only HTML5 and CSS3.
> No JavaScript. No frameworks. Just structure and styling.

─────────────────────────────────────────────

OVERVIEW

──────────────────────────────────────────────

Electrochip is a static e-commerce interface designed to demonstrate
how online store layouts are built using core web technologies.

This project focuses entirely on:

• Semantic HTML structure
• Modern CSS layout techniques
• Visual hierarchy for product presentation
• Responsive design without scripting
• Clean, readable frontend architecture

There is no dynamic behavior — this is intentional.

The goal is to master layout before logic.

──────────────────────────────────────────────

TECH STACK

──────────────────────────────────────────────

HTML5    → Page structure and content
CSS3     → Styling, grid, flexbox, responsiveness

No JavaScript.
No libraries.
No build tools.

Runs directly in any modern browser.

──────────────────────────────────────────────

PROJECT STRUCTURE

──────────────────────────────────────────────

Electrochip/
│
├── index.html        # Main storefront page
├── style.css         # All styling rules
│
└── assets/
├── images/       # Product / banner visuals
└── icons/        # UI graphics

Simple. Transparent. Easy to understand.

──────────────────────────────────────────────

FEATURES

──────────────────────────────────────────────

[✔] Static storefront layout
[✔] Product grid using CSS Flexbox / Grid
[✔] Navigation bar and hero section
[✔] Clean typography and spacing system
[✔] Responsive behavior using media queries
[✔] No dependency on scripts or frameworks

Everything you see is achieved using layout techniques only.

RUN LOCALLY

──────────────────────────────────────────────

Clone the repository:

```
git clone https://github.com/Dev-Chandan404/Electrochip.git
```

Open the folder:

```
cd Electrochip
```

Launch the site:

```
open index.html
```

Or double-click `index.html`.

That's it.

LEARNING OBJECTIVES

──────────────────────────────────────────────

This project is ideal for understanding:

• How real websites are structured before JS is added
• CSS layout systems (Flexbox / Grid)
• Responsive design fundamentals
• Separation of content and presentation
• Building UI without relying on frameworks

If you can build this without JavaScript,
you understand the foundation of the web.
